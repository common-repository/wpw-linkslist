A flexible replacement for the standard links widget, for wordpress 2.0.x only.

== Description ==

This widget replaces the standard Blogroll widget, and adds the following features:

* API compatibility, so the widget matches the theme
* Respects options set in the Links Manager Admin Panel

== Installation ==

This is a widget, and as such requires the widgets plugin from http://automattic.com/code/widgets

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag "Links List" widget to your sidebar where you would like it to appear

== Frequently Asked Questions ==
